# 流水 TodoList

一个移动端优先的待办事项应用，设计风格取意"流水禅意"——柔和、自然、宁静。

**[在线体验](https://liushui-todolist.netlify.app)** · **[配套文档](./docs/)**

---

## 功能

- ✅ 添加、完成、编辑、删除任务
- 🏷 三种优先级（高 / 中 / 低）
- 🔍 筛选视图（全部 / 待完成 / 已完成）
- 💾 本地存储，刷新不丢失
- 📱 响应式设计，手机和电脑都能用

## 设计风格

薄荷绿渐变配暖白底，辅以柔和的动效和圆角系统。没有多余的装饰，每个元素都有它存在的理由。

## 技术栈

| 层级 | 技术 |
|------|------|
| 结构 | HTML5 |
| 样式 | CSS3（Flexbox / CSS 变量 / @keyframes） |
| 交互 | 原生 JavaScript（无框架依赖） |
| 存储 | LocalStorage |
| 部署 | Netlify |

## 快速上手

直接用浏览器打开 `todolist.html` 即可运行。
